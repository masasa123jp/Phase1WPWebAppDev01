== Description ==
Provides weekly dog‑care advice, gacha‑style facility finder, photo uploads,
and KPI dashboard – built as a Gutenberg‑native plugin.

== Installation ==
1. Upload `roro-core` folder to `wp-content/plugins/`.
2. Activate through Plugins screen.
3. Go to **Settings › RoRo** to enter Maps & AdSense keys.
